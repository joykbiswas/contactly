# Project Name: CONTACTLY

## Description:
Contactly is a user-friendly contact management application built with the MERN stack (Mongoose, MongoDB, Next.js, Node.js) and TypeScript. Styled with Tailwind CSS, Contactly allows users to effortlessly create, view, edit, and delete contacts, all displayed on an intuitive homepage.
